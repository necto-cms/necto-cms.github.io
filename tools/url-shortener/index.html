Secure URL Shortener

Create lightning-fast standard short links, or use our zero-knowledge encrypted mode to share sensitive URLs without trusting the server.

Lightning Fast

Powered by Next.js and Cloudflare Workers at the edge, reducing latency to milliseconds worldwide.

Zero-Knowledge

In encrypted mode, AES-256-GCM encryption happens completely in your browser. Complete privacy.

URL Hashes

The decryption key is stored exclusively in the URL fragment (#hash), which browsers never transmit to servers.

How Encryption Works

  • 1
    Local EncryptionYour browser uses the Web Crypto API to generate an AES-GCM key and encrypts the URL locally.
  • 2
    Blob StorageOnly the unrecognizable, encrypted payload is sent to our Cloudflare Worker and stored in the KV database.
  • 3
    Client-Side RedirectionWhen visited, our server returns a static HTML page. The page extracts the key from the `#hash`, decrypts the URL locally, and redirects you seamlessly.
